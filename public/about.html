Welcome to the Color App.

The purpose of this application is to search/retrieve & copy colors from an SQL database. Behind the scenes, The Color Api holds four-thouand and ninety-six (4096) colors to choose from.

The color pallette supplies four-thouand and ninety-six (4096) of those colors by fetching or getting the list of colors from the /colors endpoint.


Data formatted in r,g&b bring three colors colors together, each ranging between 0 & 255, as a multiple of 17.
(number) % 17 is equal to 0

When using the color-search module, ONLY colors in the database, or palette, are available to search. In addition, users can search for Hex values as long as the hex value matches the RGB value in the database.
